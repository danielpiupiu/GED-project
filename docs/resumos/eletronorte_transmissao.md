# INSTRUÇÃO PARA NUMERAÇÃO DE DOCUMENTOS DE SUBESTAÇÃO

## OBJETIVO

Esta instrução tem por objetivo padronizar a numeração dos documentos técnicos de projetos civis, de eletromecânicos e de fabricantes de Subestações da Eletrobrás Eletronorte.

## FORMAÇÃO DOS NÚMEROS

São apresentadas neste item as formações dos números dos documentos de projeto e de fornecedores.

As identificações do Sistema e do Empreendimento deverão ser definidas pela Eletrobrás Eletronorte.

### Documentos de Projeto

Formação: AAA-BBB-CCDEE-FF-GG

| Sigla | Definição                                                                                 |
| :---: | :---------------------------------------------------------------------------------------- |
|  AAA  | Sistema de Transmissão ou Padronização                                                    |
|  BBB  | Empreendimento ou Subestação                                                              |
|  CC   | Definição do Assunto                                                                      |
|   D   | Definição da Área do Projeto. Para o Arquivo Mãe, este dígito significa o nível de tensão |
|  EE   | Número Sequencial                                                                         |
|  FF   | Identificação da Etapa de Projeto                                                         |
|  GG   | Identificação do Tipo de Documento                                                        |

Para a etapa Consolidação, os dígitos (**FF**) são **00**.

Os documentos **PES-090** deverão ter os dígitos **00** (**FF**).

Para os documentos técnicos solicitados pela Diretoria de Operação, os 2 dígitos (**FF**) são **99**.

## INDICAÇÃO DE REVISÃO DE DOCUMENTOS

Este item é aplicável para documentos de projetos e de fabricantes.

Na fase de aprovação de qualquer documento técnico, a indicação das revisões deverá ser alfanumérica. No primeiro envio, todo documento técnico deverá receber a indicação “A”, no campo destinado a anotação das revisões. Caso sejam necessárias outras revisões antes do documento ser considerado “APROVADO”, as mesmas deverão ser indicadas com “B”, “C”, “D”, etc., sequencialmente.

Após a primeira aprovação, o histórico de revisões alfabéticas deve ser eliminado e o documento técnico deverá receber a indicação “00” (zero zero) no campo destinado à anotação da emissão inicial.

Caso sejam necessárias outras revisões do documento após a primeira aprovação, as mesmas deverão ser indicadas como “00A”, “00B”, “00C”, etc., até a nova aprovação, quando o documento deverá receber a indicação “01” (zero um), no campo destinado à anotação das revisões, repetindo este procedimento sempre que se fizer necessário.

Para mais informações, ver PES-090-01020-00-EP.

## NOMENCLATURA DE ARQUIVOS MAGNÉTICOS

### Documentos de Projeto

#### Desenhos em AutoCad

Os nomes dos arquivos de desenhos (plantas, cortes, diagramas, etc) de um determinado projeto serão os próprios números dos documentos, com seus complementos, sendo indicada a revisão.

Exemplos:

Documentos em Aprovação

- AAA-BBB-02400-01-RA.dwg (Locação das Fundações e Canaletas)
- AAA-BBB-02400-01-R00A.dwg (Locação das Fundações e Canaletas)
- AAA-BBB-02400-01-R01A.dwg (Locação das Fundações e Canaletas)
- AAA-BBB-02400-01-R10A.dwg (Locação das Fundações e Canaletas)

Documentos Aprovados

- AAA-BBB-02000-01-R01.dwg (Planta de Localização)
- AAA-BBB-02000-01-R10.dwg (Planta de Localização)

#### Arquivos de Texto e de Planilhas

O nome dos arquivos de texto e de planilhas será o próprio número do documento, com seu complemento, sendo indicada a revisão.
